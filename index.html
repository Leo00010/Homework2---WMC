Photos

Image format: JPG (JPEG), which is good for photos. It has lossy compression, making it suitable for high-quality, small file sizes. Use HDR for better dynamic range in your photos.

Graphics

Image format: PNG, great for graphics and transparency. PNG supports lossless compression, which means no quality is lost. HDR can be used for more vivid images in high-quality graphics.

Sound

Audio format: MP3, commonly used for streaming. It is compressed, so it's ideal for quick loading times while maintaining decent quality.

GIF Animation

GIF format: Used for animations and simple graphics. It supports transparency and looping, making it ideal for small, repeating animations.

Video

Video format: Embedding from YouTube is the best way to show a video. It automatically adjusts for different screen sizes, and you can include captions and subtitles.
